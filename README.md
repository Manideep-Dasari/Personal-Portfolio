# Responsive Personal Portfolio Website

## Project Overview
This project is a responsive personal portfolio website built to showcase my skills, projects, and professional experience. The goal of this project is to
apply modern web design principles while ensuring accessibility, responsiveness, and clean code structure.

The website is designed to work seamlessly across mobile, tablet, and desktop devices and follows semantic HTML5 standards.

---

## Features
- Fully responsive design (mobile, tablet, desktop)
- Semantic HTML5 structure
- CSS Grid and Flexbox layouts
- Accessible navigation and form elements
- Mobile-friendly hamburger navigation menu
- Contact form with basic HTML5 validation
- Light/Dark mode toggle
- Clean and maintainable code structure

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (basic interactivity)
- Git & GitHub for version control

```

```
2. Open the project folder in VS Code.
3. Open `index.html` in any modern web browser.
   
---

## Technical Details
- **Navigation Menu:** Implemented using JavaScript to toggle visibility on mobile devices.
- **Layout:** CSS Grid is used for project layouts, and Flexbox is used for navigation and form alignment.
- **Responsive Design:** Media queries adjust layout for different screen sizes.
- **Dark Mode:** Implemented using CSS variables and JavaScript class toggling.
- **Accessibility:** Proper use of semantic tags, labels, alt attributes, and ARIA labels.

---

## Testing & Validation
- Tested on Google Chrome and Mozilla Firefox
- Verified responsive behavior using browser developer tools
- HTML5 form validation used for contact form inputs
- Manual accessibility checks for keyboard navigation and screen readability
